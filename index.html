Project Overview

The objective of this project was to develop a predictive modeling framework that estimates NHL player contract value using historical contract data, player performance metrics, demographic variables, and advanced player-value statistics.

NHL contracts are complex financial decisions influenced by scoring production, age, position, usage, team context, and prior performance. This project quantified those relationships using Python-based data preparation, feature engineering, statistical modeling, and interactive Tableau dashboarding.

The analysis combined publicly available contract data from PuckPedia with player statistics and advanced performance data from Evolving Hockey. The workflow focused on cleaning and standardizing player records, engineering contract-year features, merging performance and contract datasets, and developing models to estimate future contract outcomes.

The final deliverable is an interactive Tableau dashboard that presents a player-level contract valuation card, allowing users to compare actual contract outcomes against model-based expectations and explore the statistical profile behind each player.

Methodology

The Python workflow begins by cleaning and preprocessing contract data from PuckPedia and player performance data from Evolving Hockey. Player names, season fields, position indicators, height, weight, shooting hand, team abbreviations, and contract fields are standardized before the datasets are merged.

After preprocessing, the project engineers modeling features related to player demographics, contract-year variables, position labels, team-ending-season indicators, cap-adjusted contract measures, traditional production, time on ice, special teams usage, and advanced statistics. These features are then used to estimate future contract value and compare predicted outcomes against actual contract results.

Key Feature Groups

  • Player demographics: age, height, weight, shooting hand, position
  • Traditional statistics: games played, goals, assists, points, penalties
  • Usage statistics: 5v5 TOI, power-play TOI, penalty-kill TOI
  • Advanced metrics/stats: player-value indicators and performance components
  • Contract data: AAV, contract length, cap percentage, signing season
  • Contextual variables: team, contract year, and season-based indicators
